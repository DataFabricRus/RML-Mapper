RML Mapper
==========

Clone RML-Mapper
----------------

	git clone --recursive git@git.mmlab.be:rml/RML-Mapper.git

Build RML-Mapper
----------------

Run 

	bin/RML-Mapper


Run RML-Mapper
-------------

	bin/RML-Mapper -m <mapping_doc> -o <output_file> [-f <file_format> -tm <triples_map> -p <parameter> -g <graph>]
